# personal-sauna

## How to Run

### Docker Compose

Na root do projeto

```
  docker compose up --build
```
  
### App e Api individualmente

Na pasta EnergyApi/

```
  pip3 install -r requirements.txt
  
  python3 -m flask run
```
  
Na pasta website/personal-sauna/

```
  npm install
  
  npm start
```
